# User Management

## Adding Users
1. Navigate to the user management section.
2. Click on `Add User` button.
3. Fill in the user details (username, email, etc.).
4. Click `Save` to add the user.

## Deleting Users
1. Select the user you want to delete from the user list.
2. Click on the `Delete` button.
3. Confirm the deletion action.

# Managing Pull Requests
- Ensure you have permission to create a pull request.
- Click on `Create Pull Request` in the repository.
- Fill in the necessary details and submit the request.

# Lordo Permissions
- Define roles for user permissions.
- Allow or restrict access based on user roles.
- Regular audits of permissions recommended.
